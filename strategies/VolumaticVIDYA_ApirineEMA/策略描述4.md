# Volumatic VIDYA + Apirine Slow RSI 策略描述

## 策略概述

本策略结合了Volumatic Variable Index Dynamic Average（VIDYA）和Apirine Slow RSI两个免费指标，通过趋势确认和超跌反弹的方法进行交易。策略适用于趋势行情和震荡行情，信号清晰直观，无需复杂计算。

## 交易规则

* 目的是做顺势单（大右小右）
* 斜率计算方式：
    - (vidyaLower - vidyaLower[1] + vidyaLower[1] - vidyaLower[2] + vidyaLower[2] - vidyaLower[3]) / 3
* 只用qqeModBar进场做回调，入场方式不同

### ====== 做多规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为绿色
    - 此时不确定的趋势刚刚启动
2. **进行指标确认**：所有条件都为true
    - 是开启指标确认状态
    - qqeModBar[2] > qqeModBar[1] < qqeModBar and qqeModBar[1] < -10 (-10可以配置)
    - min(slowRsi[2], slowRsi[1], slowRsi) < 30 (30可以配置)
    - VIDYA通道下沿斜率必须大于0.0001
    - trendSpeed > trendSpeed[1]
    - dynEma斜率必须大于0.0001
3. **入场时机**：
    - 趋势确认 + 指标确认时，可以进场
    - 买入点：对应K线收盘价

#### 做多止损/止盈
- 止损设在VIDYA通道下沿
- 固定盈亏比1:2，(2可以配置)

### ====== 做空规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为红色
    - 此时不确定的趋势刚刚启动
2. **进行指标确认**：所有条件都为true
    - 是开启指标确认状态
    - qqeModBar[2] < qqeModBar[1] > qqeModBar and qqeModBar[1] > 10 (10可以配置)
    - max(slowRsi[2], slowRsi[1], slowRsi) > 70 (70可以配置)
    - VIDYA通道上沿斜率必须小于-0.0001
    - trendSpeed < trendSpeed[1]
    - dynEma斜率必须小于-0.0001
3. **入场时机**：
    - 趋势确认 + 指标确认时，可以进场
    - 买入点：对应K线收盘价

#### 做空止损/止盈
- 止损设在VIDYA通道上沿
- 固定盈亏比1:2，(2可以配置)
