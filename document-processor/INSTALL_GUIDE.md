# 安装指南

## 新增依赖

为了解决中文乱码和控件丢失问题，我们添加了新的依赖库。

## 安装步骤

1. **安装新依赖**：
```bash
npm install
```

2. **如果安装失败，手动安装**：
```bash
npm install officegen archiver
```

## 新的解决方案

### 中文乱码修复：
- 使用Buffer正确处理UTF-8编码
- 确保中文字符在XML中正确显示
- 改进了字符串编码处理

### 控件保护：
- 使用最简单的字符串替换方法
- 避免复杂的正则表达式
- 减少对XML结构的破坏

## 如果仍有问题

如果中文乱码和控件丢失问题仍然存在，我们可以：

1. **使用更专业的库**（如 `docx-templates`）
2. **改用纯文本模板**（避免Word控件）
3. **使用其他文档格式**（如HTML转PDF）

## 运行程序

```bash
npm start
```

现在应该能同时解决占位符替换、中文乱码和控件保护问题。
