# Volumatic VIDYA + Apirine Slow RSI 策略描述

## 策略概述

本策略结合了Volumatic Variable Index Dynamic Average（VIDYA）和Apirine Slow RSI两个免费指标，通过趋势确认和超跌反弹的方法进行交易。策略适用于趋势行情和震荡行情，信号清晰直观，无需复杂计算。

## 交易规则

* 目的是做顺势单（大右小右）
* 优化进出场逻辑

### ====== 做多规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为绿色
    - 此时不确定的多头趋势刚刚启动为true，空头为false
    - 每一段趋势可以有多个订单，只要符合进单条件即可
2. **判断开单**：
    - 多头趋势确认为true
    - 当前dynEma为绿色
    - VIDYA通道下沿斜率必须大于0.0001
    - 下面任意条件可入场开单
        - dynEma从红色变为绿色
            - dynEma为绿色 and dynEma[1]为红色
        - qqeModBar拐头向上
            - qqeModBar[2] > qqeModBar[1] < qqeModBar and qqeModBar[1] < -1 (-1可以配置)
3. **开单**：
    - 开仓价：close
    - 止损价：min(VIDYA通道下沿，前低)
    - 止盈价：使用固定盈亏比1:2，(2可以配置)
4. **多单止盈**：
    - 等待价格触发固定止盈位置即可
    - 趋势结束仓位仍在，仍然按照原来的多单止盈止损逻辑等待最终结果
5. **多单止损**：
    - 如果 (当前价格 - 开仓价) / 开仓价 >= 0.5%，设置保本止损，修改仓位止损位置到保本位置（0.5%可以配置）
    - 如果 当前VIDYA通道下沿 > 当前限价止损价，修改仓位止损位置到当前VIDYA通道下沿
    - 开单后当遇到 (qqeModBar[2] < qqeModBar[1] > qqeModBar and qqeModBar[1] > 20 (20可以配置)) 或者 盈亏比达到1:1时(1可配置)，立即平仓50%，进入后半仓平仓模式
        - 后续每根k线都要更新限价止损为：max(当前止损，前低)


### ====== 做空规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为红色
    - 此时不确定的空头趋势刚刚启动为true，多头为false
    - 每一段趋势可以有多个订单，只要符合进单条件即可
2. **判断开单**：
    - 空头趋势确认为true
    - dynEma为红色
    - VIDYA通道上沿斜率必须小于-0.0001
    - 下面任意条件可入场开单
        - dynEma从绿色变为红色
            - dynEma为红色 and dynEma[1]为绿色
        - qqeModBar拐头向下
            - qqeModBar[2] < qqeModBar[1] > qqeModBar and qqeModBar[1] > 1 (1可以配置)
3. **开单**：
    - 开仓价：close
    - 止损价：max(VIDYA通道上沿，前高)
    - 止盈价：使用固定盈亏比1:2，(2可以配置)
4. **空单止盈**：
    - 等待价格触发固定止盈位置即可
    - 趋势结束仓位仍在，仍然按照原来的空单止盈止损逻辑等待最终结果
5. **空单止损**：
    - 如果 (开仓价 - 当前价格) / 开仓价 >= 0.5%，设置保本止损，修改仓位止损位置到保本位置（0.5%可以配置）（删除该功能）
    - 如果 当前VIDYA通道上沿 < 当前限价止损价，修改仓位止损位置到当前VIDYA通道上沿
    - 开单后当遇到(qqeModBar[2] > qqeModBar[1] < qqeModBar and qqeModBar[1] < -20 (-20可以配置))时 或者 盈亏比达到1:1时(1可配置)，立即平仓50%，进入后半仓平仓模式
        - 后续每根k线都要更新限价止损为：min(当前止损，前高)
