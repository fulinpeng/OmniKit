# Volumatic VIDYA + Apirine Slow RSI 策略描述

## 策略概述

本策略结合了Volumatic Variable Index Dynamic Average（VIDYA）和Apirine Slow RSI两个免费指标，通过趋势确认和超跌反弹的方法进行交易。策略适用于趋势行情和震荡行情，信号清晰直观，无需复杂计算。

## 交易规则

* 目的是做顺势单（大右小右）
* 斜率计算方式：
    - (vidyaLower - vidyaLower[1] + vidyaLower[1] - vidyaLower[2] + vidyaLower[2] - vidyaLower[3]) / 3
* 可配置分批止盈的比例
* 同样的，是否设置保本其实区别不大，因为有移动止损代替了保本
    - 也可能时保本被移动止损覆盖了
* 两种开单方式不行，只用qqeModBar进场做回调
* 不设佣金，aster可以达到46%收益，胜率52%

### ====== 做多规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为绿色
    - 此时不确定的趋势刚刚启动
2. **进行指标确认**：所有条件都为true
    - 是开启指标确认状态
    - high < 前高
    - VIDYA通道下沿斜率必须大于0.0001
    - 最近5根qqeModBar向上拐头，并且最小值必须小于-10具体如下：
        - qqeModBar[4] > qqeModBar[3] > qqeModBar[2] < qqeModBar[1] < qqeModBar and qqeModBar[2] < -10(-10可以配置)

3. **入场时机**：
    - 趋势确认 + 指标确认时，可以进场
    - 买入点：对应K线收盘价

#### 做多止损/止盈
- 止损设在最近的低点
- 固定盈亏比1:1，(1可以配置)

### ====== 做空规则 ======

#### 入场条件
1. **趋势确认**：
    - 最新的VIDYA通道为红色
    - 此时不确定的趋势刚刚启动
2. **进行指标确认**：所有条件都为true
    - 是开启指标确认状态
    - low > 前低
    - VIDYA通道上沿斜率必须小于-0.0001
    - 最近5根qqeModBar向下拐头，并且最大值必须大于10具体如下：
        - qqeModBar[4] < qqeModBar[3] < qqeModBar[2] > qqeModBar[1] > qqeModBar and qqeModBar[2] > 10(10可以配置)
3. **入场时机**：
    - 趋势确认 + 指标确认时，可以进场
    - 买入点：对应K线收盘价

#### 做空止损/止盈
- 止损设在最近的高点
- 固定盈亏比1:1，(1可以配置)
