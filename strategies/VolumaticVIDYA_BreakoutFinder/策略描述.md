# Volumatic VIDYA + Breakout Finder 剥头皮策略描述

## 策略概述

本策略结合了Volumatic Variable Index Dynamic Average（VIDYA）和Breakout Finder两个免费指标，通过突破方法进行剥头皮交易。策略适用于外汇、黄金、加密货币等多种交易品种。

## 指标组合

### 1. Volumatic Variable Index Dynamic Average (VIDYA)
- **创作者**：Big Beluga
- **功能**：作为市场趋势指导指标
- **参数设置**：
  - Distance Factor: 4
  - 关闭Labels显示
- **信号说明**：
  - **绿色箭头**：趋势从看跌转为看涨（看涨反转信号）
  - **红色箭头**：趋势从看涨转为看跌（看跌反转信号）

### 2. Breakout Finder
- **创作者**：Lonesome the blue
- **功能**：识别市场突破机会
- **信号说明**：
  - **蓝色三角形**：看涨突破，买入信号
  - **红色三角形**：看跌突破，卖出信号

## 策略设置

### 指标加载步骤
1. 在TradingView中搜索 **"Volumatic Variable Index Dynamic Average"**
2. 选择创作者为 **Big Beluga** 的指标
3. 在Input标签中，将 **Distance Factor** 设置为 **4**
4. 在Style标签中，取消勾选 **Labels** 选项
5. 点击确认保存设置
6. 搜索 **"Breakout Finder"**
7. 选择创作者为 **Lonesome the blue** 的指标
8. 添加到图表中

## 交易规则

### ====== 做多规则 ======

#### 入场条件（满足所有条件）
1. **VIDYA确认**：
   - Volumatic VIDYA指标显示绿色箭头
   - 表示趋势从看跌转为看涨（看涨反转）
2. **Breakout Finder确认**：
   - Breakout Finder指标显示蓝色三角形
   - 表示看涨突破发生
3. **K线确认**：
   - 形成的K线必须是看涨K线（收盘价 > 开盘价）
4. **入场时机**：
   - 在看涨K线收盘后下单

#### 做多止损
- 止损设在当前10根k线的最高点

#### 做多止盈
- 盈亏比：1:2
- 即：如果止损距离为X，则止盈目标为入场价 + 2X

---

### ====== 做空规则 ======

#### 入场条件（满足所有条件）
1. **VIDYA确认**：
   - Volumatic VIDYA指标显示红色箭头
   - 表示趋势从看涨转为看跌（看跌反转）
2. **Breakout Finder确认**：
   - Breakout Finder指标显示红色三角形
   - 表示看跌突破发生
3. **K线确认**：
   - 形成的K线必须是看跌K线（收盘价 < 开盘价）
4. **入场时机**：
   - 在看跌K线收盘后下单

#### 做空止损
- 止损设在当前10根k线的最低点

#### 做空止盈
- 盈亏比：1:2
- 即：如果止损距离为X，则止盈目标为入场价 - 2X
