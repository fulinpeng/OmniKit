# Volumatic VIDYA + Apirine Slow RSI 策略描述

## 策略概述

本策略结合了Volumatic Variable Index Dynamic Average（VIDYA）和Apirine Slow RSI两个免费指标，通过趋势确认和超跌反弹的方法进行交易。策略适用于趋势行情和震荡行情，信号清晰直观，无需复杂计算。


## 指标组合

### 1. Volumatic Variable Index Dynamic Average (VIDYA)
- **功能**：动态趋势线，实时追踪市场波动，精准标记支撑阻力位，结合成交量分析买卖压力
- **核心逻辑**：
  - 通过动态趋势线VIDYA追踪市场波动
  - 结合成交量分析买卖压力
  - 趋势方向一目了然，识别起来毫不费力
- **优势**：相比SuperTrend指标，VIDYA对趋势跟踪更加丝滑，不会在震荡行情中反复提示多空开仓

### 2. Apirine Slow RSI
- **功能**：抄底摸底绝密指标，慢速RSI指标
- **核心逻辑**：
  - 显示RSI线slowRsiEMA7线和slowRsiEMA30线
  - RSI线低于50为超跌区域
  - slowRsiEMA7上穿slowRsiEMA30形成金叉为买入信号
- **参数设置**：
  - Smoothing: 6（默认）
  - RSI Length: 14（默认）
  - SlowRSI EMA Length: 9（默认）
  - ema1Len：7
  - ema2Len：30

## 交易规则

* 目的是做顺势单（大右小右）
* 斜率计算方式：
    - (vidyaLower - vidyaLower[1] + vidyaLower[1] - vidyaLower[2] + vidyaLower[2] - vidyaLower[3]) / 3
* 本次优化参数请看aster2.png，胜率41.5
    - 不开保本，不开分批止盈盈利280%，但是曲线不平滑没有明显上升趋势，回撤27%
    - 不开保本，开启分批止盈胜率65%, 盈利180%,曲线明显上升趋势，回撤30%

### ====== 做多规则 ======

#### 入场条件
1. **趋势确认**：
   - 最新的VIDYA通道为绿色
   - 此时不确定的趋势刚刚启动
   - dynEmaColor 从红色变为绿色，开启指标确认
    - dynEmaColor 从绿色变为红色，关闭指标确认
    - 止盈或者止损也闭指标确认
2. **进行指标确认**：
   - 是开启指标确认状态
   - trendSpeed 柱子颜色为绿色
   - VIDYA通道下沿斜率不能向下，大于0.0001
   - close < VIDYA通道上沿
   - not(slowRsi >= 60 and slowRsiEMA7 >= 60 and slowRsiEMA30 >= 60)  （60可配置）

3. **入场时机**：
   - 趋势确认 + 指标确认时，可以进场
   - 买入点：对应K线收盘价

#### 做多止损
- 止损设在 VIDYA通道下沿
- 固定盈亏比1:3，后续不再修改最大止盈点(3可以配置)

#### 做多止盈
- 开启分批止盈的话：
    - 如果达到 [close >= max(VIDYA通道上沿, 前高) or 达到1：1.5盈亏比]，先平仓50%，然后止损价格改为 max(前低, 原来的止损价格）：
    - 如果已经首次止盈，然后后续持续修改止损逻辑: 
        - 如果 VIDYA通道下沿 > 原止损价格，那么判断 close < VIDYA通道下沿 立即市价止损平仓
        - 如果 VIDYA通道下沿 <= 原止损价格，止损价格还是保持原来的止损价格限价止损
        - 如果 dynEma 斜率 < -0.002，立即全部平仓
- 盈亏比达到1:1时，移动止损点到保本位置（开仓价格*1.001）
- 趋势尾端：多头趋势反转视为尾端（VIDYA通道绿色转红色）
